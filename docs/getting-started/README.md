# Getting Started with MCP

Welcome to the Model Context Protocol! This guide will help you get started quickly.

## What is MCP?

The Model Context Protocol (MCP) is a standardized protocol for AI models to interact with external systems. It enables seamless integration between AI models and tools, resources, and prompts.
