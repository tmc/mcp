# Experimental MCP Commands

This directory contains experimental tools and commands for the Model Context Protocol (MCP) that are not yet ready for production use.

## Current Experimental Tools

Various experimental commands are being developed here. Tools may graduate to the main cmd/ directory when they become stable.

## Related Experimental Projects

- **mcp9p**: Plan9-inspired namespace system for MCP service discovery and management (see `../mcp9p/`)

## Contributing

When adding new experimental commands:
1. Create a new directory under exp/cmd/
2. Add a README.md explaining the tool
3. Include examples and use cases
4. Document why it's experimental and what's needed for stability

## Stability Notice

⚠️ **WARNING**: These tools are experimental and their APIs may change without notice. They should not be used in production systems.
