# ctx-go-src

A simple tool to dump Go package source code in txtar format.

## Usage

```bash
# Dump a local package
ctx-go-src ./cmd/mytool

# Dump a remote package
ctx-go-src github.com/user/package

# Dump standard library package
ctx-go-src fmt

# Include subdirectories
ctx-go-src -r ./pkg

# Include test files
ctx-go-src -tests ./cmd/mytool
```

## Features

- Dumps Go source files in txtar format
- Automatically escapes existing txtar headers in source files
- Can fetch remote packages using go module resolution
- Handles standard library packages
- Optional recursive directory traversal
- Optional test file inclusion

## Example

```bash
$ ctx-go-src ./exp/cmd/ctx-go-src/test
# Go source dump for /path/to/test
# Generated by ctx-go-src

-- example.go --
package test

import "fmt"

func Example() {
    fmt.Println("Hello")
}
```
