# MCP Experimental Documentation

This directory contains documentation for the experimental MCP tools and features.

## Documents

### TOOL_GRAPH.md
A comprehensive graph showing all MCP tools, their relationships, and dependencies. Includes:
- Complete tool categorization
- Dependency relationships
- Future tool concepts
- Mermaid diagrams for visualization

### TOOL_OVERVIEW.md
A simplified overview focusing on the most commonly used tools and workflows. Includes:
- Primary tool flows
- Common use cases
- Quick start paths
- Tool selection guide

## Purpose

These documents serve to:
1. Provide a birds-eye view of the MCP tool ecosystem
2. Help developers choose the right tools for their tasks
3. Show how tools work together in common workflows
4. Document future development directions

## Contributing

When adding new tools to the experimental directory:
1. Update TOOL_GRAPH.md with the new tool's category and dependencies
2. Add practical examples to TOOL_OVERVIEW.md if it's a commonly used tool
3. Consider creating tool-specific documentation if needed

## Visual Aids

Both documents include Mermaid diagrams that can be rendered in:
- GitHub markdown preview
- Most modern markdown editors
- Documentation sites that support Mermaid

To view the diagrams, simply open the markdown files in any Mermaid-compatible viewer.
